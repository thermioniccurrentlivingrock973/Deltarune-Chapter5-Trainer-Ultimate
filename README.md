## 🎮 Using the trainer features

The interface displays several options once the program loads. Ensure the game is running before you toggle these options.

### Adjusting stats
Navigate to the Stats tab. You will see fields for Attack, Defense, and Speed. Type the desired number into the text box and click the Update button to push the values to the game.

### Managing gold
Select the Gold tab to update your wallet. Enter the amount of gold you want and press the Set Gold button. The game updates the value instantly.

### Enabling god mode
Open the Misc tab to find the God Mode toggle. Click the box to turn this feature on. Your character will not take damage while this setting remains active. Click the box again to turn the effect off.

## 📋 Frequently asked questions

Does this tool change my game files permanently?
No. The trainer only modifies values in the computer's memory while the game runs. If you close the game and the trainer, your stats will return to their original values upon your next restart unless you save your progress in the game menu.

Will this get me banned from online play?
Deltarune is a single-player game. You do not need to worry about online bans. Use the tool freely to enjoy your game experience.

Can I customize the hotkeys?
Yes. Go to the Settings tab to bind buttons on your keyboard to specific trainer functions. This allows you to toggle features without leaving the game screen.

What if the trainer crashes?
Close both the game and the trainer. Reopen the game, reach the main menu, and then launch the trainer again. This sequence prevents conflicts between the two programs.
